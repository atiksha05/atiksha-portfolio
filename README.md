## Getting Started

```bash
# Clone the repository
git clone https://github.com/atiksha05/atiksha-portfolio.git
cd atiksha-portfolio

# Install dependencies
npm install

# Run locally
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

```bash
# Production build
npm run build
npm start
```

> **Note:** Large video assets (`public/videos/`) are tracked with [Git LFS](https://git-lfs.github.com/). Run `git lfs install` after cloning if videos are missing.
